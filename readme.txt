[実装内容]
α版（～10/13）
・ヒップドロップ
空中でのみ使用できる技です。当たった対象が地形か敵で効果が変わります。
地形に当たると、カメラが一瞬引くことによって、周囲を確認できます。
敵に当たると、プレイヤーが上に跳ねて、２段目のジャンプのフラグがリセットされます。

β版途中経過（～10/18）
・プレイヤーモデル
アデリーペンギンのヒナ。
モーション込みでだいぶリメイク元寄りです。
・空中攻撃（仮）
プレイヤーが向いている方向の斜め下目掛けて弾を飛ばします。
攻撃中は落下速度が下がります。
これから地形や敵との当たり判定を実装予定です。

β版（～10/20）
・ダメージブロック
赤いブロック。
当たると敵と同じようにダメージ受けます。
ヒップドロップでダメージを回避できます。
